# GWALLUP 🌅

> *Ride Into the Future*

![Mason Riding Into the Sunset](mason_sunset.png)

---

## What Is This

GWALLUP is a frontier AI protocol — and a website — built around one image: Mason, riding a horse into the AI sunset. It's less a product and more a philosophy. Move toward the unknown. No map. Just momentum.

The site is a single-page experience with four thematic tabs:

| Tab | Icon | What It Is |
|-----|------|-----------|
| **The Frontier** | 🌅 | Hero landing — the pitch, the vibe, the mission |
| **The Trail** | 🐴 | The story of Mason and the GWALLUP origin |
| **The Posse** | 🤠 | The team behind the ride |
| **The Horizon** | ✦ | Roadmap — phases I through IV and beyond |

---

## Stack

- Pure HTML/CSS/JS — zero dependencies, zero frameworks
- Google Fonts: `Playfair Display`, `Space Mono`, `Inter`
- Single file: `index.html`
- Image: `mason_sunset.png` (AI-generated, cinematic silhouette)

---

## Aesthetic

- **Palette:** Deep midnight purple → sunset orange → pale gold
- **Typography:** Serif display (Playfair) + monospace (Space Mono) + clean body (Inter)
- **Mood:** Cinematic western × AI future — dark, warm, epic
- **Accent:** Sunset orange `#E8622A` as the primary throughline

---

## Local Development

No build step required. Just open the file:

```bash
open index.html
```

Or serve it locally:

```bash
npx serve .
# → http://localhost:3000
```

---

## Deploy

Push to GitHub and connect to Vercel for instant deployment:

```bash
git add .
git commit -m "ride"
git push
```

Then import the repo at [vercel.com/new](https://vercel.com/new). It's a static site — deploys in seconds.
